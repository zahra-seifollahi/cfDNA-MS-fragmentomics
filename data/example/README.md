# Synthetic example fragment files

This folder contains synthetic `.frag.gz` fragment files and sample metadata for testing the cfDNA-MS fragmentomics pipeline.

These files are artificial example data and do not contain real patient information.

Files:
- `fragments/`: synthetic fragment files
- `example_sample_metadata.csv`: sample group metadata
- `Healthy`, `Remission`, and `Relapse` are fictional labels used only for testing.
